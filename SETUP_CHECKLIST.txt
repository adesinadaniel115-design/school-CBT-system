SETUP CHECKLIST (LOCAL, XAMPP/LARAVEL)

1) Apache DocumentRoot
   - Point to: C:/xampp/htdocs/school-cbt/public
   - File: C:/xampp/apache/conf/httpd.conf

2) (Optional) Local Domain
   - Add vhost in: C:/xampp/apache/conf/extra/httpd-vhosts.conf
   - Add hosts entry in: C:/Windows/System32/drivers/etc/hosts

3) Environment File
   - Create .env from .env.example
   - Set APP_URL=http://localhost (or your local domain)

4) PHP Dependencies
   - Run: composer install

5) Database
   - Ensure MySQL is running
   - Run: php artisan migrate

6) Cache/Config After .env Changes
   - Run: php artisan config:clear
   - Run: php artisan cache:clear

7) Verify App
   - Visit: http://localhost

Notes:
- Do NOT use php artisan serve when Apache is already serving the app.
- If you see directory listing, DocumentRoot is wrong.
- If you see vendor/autoload.php missing, run composer install.
